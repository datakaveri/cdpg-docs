# Licensing

IUDX is built and released as a **Digital Public Good**, giving governments and organizations a zero-cost platform they can adopt, deploy, customize, and fully own — with no licensing fees.

## Open-source components

All software used to build and deploy IUDX is open-source licensed and used in its original form. The IUDX services themselves are released under the **GNU Affero General Public License (AGPL)**, and the third-party components in the stack are used under their respective open-source licenses (Apache 2.0, BSD, SSPL/Elastic License, PostgreSQL License, AGPL, and others). See the [Technology Stack](../technical/technology-stack.md) for the components used.

## What this means for adopters

- **No licensing cost** — adopt and run the platform freely.
- **Full ownership and control** — governments and organizations retain sovereignty over their own deployments and data.
- **Transparency and adaptability** — the source is open for inspection, extension, and contribution.
- **Vendor-neutral** — no lock-in to a single provider.

The source code is available on GitHub at [github.com/datakaveri](https://github.com/datakaveri).

For questions, contact **info@cdpg.org.in**.
