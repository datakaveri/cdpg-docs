## Spotlight: Improving farmer loans using Agri-Data

A representative end-to-end flow shows how access control and governance come together:

1. **Data pertaining to farmer credit** is published with appropriate terms and consent.
2. A **loan application and credit-assessment backend** consumes the data through controlled, policy-based, consent-driven access.
3. Every access is **authorized and logged**, so the data provider retains full control over what is shared and with whom.

The result: better-targeted farmer loans while keeping sensitive data under the providers' control.
